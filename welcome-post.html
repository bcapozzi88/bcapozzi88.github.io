Welcome Post

by Brian Capozzi
in Posts on 2014-09-23

I've hopped on the bandwagon and created a blog to discuss coding and data related projects that I've recently started working on. I think this is a nice way to aggregate things that I've learned, and the blog format also offers the (small) possibility that others may find interesting reading material.

I've created this blog using a combination of Pelican, Markdown, and GitHub Pages. Pelican is written primarily in Python, and is a fairly easy to use static site generator, geared towards blogging, which conveniently lets you write content using Markdown. Since I have no html experience, Markdown is ideal since it allows one to type web content using simple syntax, and have the content automatically converted to html. GitHub pages offers freely posted and easily hosted webpages.

I primarily used the two sites below (along with the three links above) as reference for making this blog. They both have easily readable instructions for getting everything up and running.

  • http://martinbrochhaus.com/pelican2.html
  • http://openjerseycity.org/blog/technical/2014/06/20/Pelican-On-GitHub-Pages/

This blog is proudly powered by Pelican, which takes great advantage of Python.

Fresh is a responsive theme designed by jsliang and contributors. Special thanks to Smashing Magazine and Twitter Bootstrap.
